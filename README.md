# 🚀 Space Explorer Dashboard

A Full Stack Web Application built for the CAPS Recruitment Task.

## 📌 Features

- NASA APOD API
- Mars Rover Photos
- Near Earth Objects
- Favorites CRUD
- MongoDB Database
- React Dashboard
- QA Documentation

## 🛠 Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB

### Frontend
- React
- Vite

### Tools
- Git
- GitHub
- Postman
